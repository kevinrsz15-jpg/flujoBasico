
# ¿Qué es una Base de Datos Relacional?
   Una base de datos relacional es un sistema que almacena información de manera organizada en tablas. Estas tablas se relacionan entre sí usando claves, permite mantener los datos ordenados, evitar duplicidad y facilitar su consulta y actualización.

# ¿Qué es una Tabla en el contexto de Bases de Datos Relacionales?
   Una tabla es la estructura principal donde se guardan los datos dentro de una base de datos relacional. Representa una entidad específica, como usuarios o productos, y organiza la información en columnas y renglones.

# ¿Qué es una Relación en Bases de Datos Relacionales?
   Una relación es la conexión lógica entre dos o más tablas. Se utiliza para vincular información relacionada, permitiendo que los datos de una tabla dependan o se asocien con los datos de otra.

# ¿Qué representa una Columna en Bases de Datos Relacionales?
   Una columna representa una característica o atributo de la información almacenada. Define el tipo de dato que se guardará, como números, texto o fechas, y mantiene la consistencia de los valores.

# ¿Qué representa un Renglón en Bases de Datos Relacionales?
   Un renglón representa un registro completo dentro de una tabla. Contiene todos los datos correspondientes a una sola entidad, como la información completa de una persona o un producto.

# ¿Qué es un Motor de Base de Datos Relacional?
   Es el software que permite crear, administrar y manipular una base de datos relacional. Se encarga de procesar consultas, guardar información, controlar accesos y asegurar la integridad de los datos.

# ¿Cuáles son los Motores de Bases de Datos Relacionales más Populares?
   Algunos de los motores más utilizados son MySQL, PostgreSQL, Oracle Database, Microsoft SQL Server y MariaDB, debido a su estabilidad, seguridad y amplio uso en aplicaciones reales.

# ¿Cuál es el Motor de Base de Datos Ligero que no Necesita un Servidor?**
   SQLite es un motor de base de datos ligero que funciona sin servidor. Se integra directamente en las aplicaciones y es ideal para proyectos pequeños o aplicaciones locales.

# ¿Qué significa SQL en el contexto de Bases de Datos Relacionales?
   SQL significa *Structured Query Language* y es el lenguaje estándar utilizado para crear tablas, insertar datos, realizar consultas y administrar bases de datos relacionales.

# ¿Cuáles son los Elementos de un Sistema de Información Basado en Bases de Datos Relacionales?
    Un sistema de información incluye el hardware, el software, la base de datos, el motor de base de datos, los usuarios y los procedimientos que permiten gestionar y utilizar la información.
